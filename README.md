# Premium Aesthetic Digital Clock

A futuristic, highly-aesthetic responsive digital clock dashboard built using React, Tailwind CSS principles, and fluid animations. Fully responsive with glassmorphic cards, dynamic time-based greetings, and custom sub-components.

## 🌟 Features

- **Ultra-Precise Live Clock:** Real-time Hours, Minutes, and Seconds tracking using a highly optimized and stable `setInterval` React hook.
- **Fluid Digit Animations:** Smooth, high-end vertical slide and fade interactions when digits change, leveraging `framer-motion` without layout jitter.
- **Glassmorphism Design:** A meticulously crafted frosted-glass central dashboard with 20px blurs, complex layered inner shadows, and a deep-space `#0f172a` backdrop.
- **Ambient Moving Blobs:** Three massive, beautifully blurred CSS gradient orbs that drift infinitely in the background for a premium, dynamic feel.
- **Dynamic Greeting:** A smart, time-aware heading that transitions seamlessly between "Good Morning", "Good Afternoon", "Good Evening", and "Late Night, Keep Grinding ✨".
- **Live Network Badge:** An active tracking module that uses `navigator.onLine` to display real-time connection status with a glowing pulse indicator.
- **12H / 24H Toggle:** Instantly switch between time formats with a styled toggle button, revealing an elegantly sliding, pulsing AM/PM pill indicator.
- **Personalized Branding:** Features a custom, floating animated footer: *Developed by Vedraj Singh*, styled with elegant letter spacing and neon glows.

## 🚀 Installation & Setup

Want to run this premium digital clock on your local machine? It takes less than 2 minutes:

1. **Clone the repository:**
   ```bash
   git clone <your-repo-link>
   cd react-digital-clock
   ```

2. **Install the dependencies:**
   Make sure you have Node.js installed, then run:
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```

4. **View the masterpiece:**
   Open your browser and navigate to `http://localhost:5173/` (or the port specified by Vite in your terminal).
